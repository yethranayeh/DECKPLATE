# DECKPLATE

Shorthand package that acts as a wrapper for [`@deckplate/cli`](https://www.npmjs.com/package/@deckplate/cli).

It simply exists as a wrapper because I initially thought I could somehow have `npx deckplate` as an alias that could run **@deckplate/cli**

```sh
npx deckplate init # npx @deckplate/cli init
npx deckplate add button # npx @deckplate/cli add button
```

All the _actual_ source code and documentation lives in main repo:
[DECKPLATE](https://github.com/yethranayeh/DECKPLATE)
